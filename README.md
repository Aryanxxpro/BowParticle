# Features
<p>These are some of the most important plugin teatures:</p>

- UI menu for easy particle selection
- Customizable ticks between each particle
- Customizable prefix

# Particles
<p>This is a list of plugin particles:</p>

- Flame
- Water
- Heart
- Smoke
- Portal
- LavaDrip
- WaterDrip
- Redstone
- SnowballPoof
- AngryVillager
- HappyVillager
- EnchantmentTable
